# What is xk-toolbox?
Xk-Toolbox is just a library of stuff.
It contains a library of different modules and styles that I stored/will store for other projects.

It contains:
 - Nodejs Modules/ Js Modules
 - CSS Stylesheets
 - json config files
-  font files
-  other
## What does it do?
Well, it does not do anything aside for being a library for js/ nodejs and css styles.
The purpose of this package is so that I can store any modules/ styles that I might use later.
This one should be more organized than my other packages.

## How do I use it?
Since I am using thi package to store my styles and themes, I would not recommend installing this package into your projects. But if you wanted to, then you can do one of the following:
- Installing into your nodejs project:
  - I use visual studio to install packages on my projects, you can learn how to that through the link [`here`](https://learn.microsoft.com/en-us/visualstudio/javascript/npm-package-management?view=vs-2022)
  - To learn how to install them locally, you can checkout this link here (Make sure you have the latest version of NodeJs installed on your operating system.) :arrow_right: [NPM Documentation](https://docs.npmjs.com/downloading-and-installing-packages-locally)

If you want to access the stylesheets, you can use the copy & paste the link below (just replace the `<version>` with the latest version of the npm package and the `<theme>`[^1])
  ``` 
  <link rel="stylesheet" href="https://unpkg.com/xk-toolbox@<version>/<theme>/style.css"/>
  ```

  ```
  <script src="https://unpkg.com/xk-toolbox@1.0.1/lib/<module-name>/index.js"></script>
  ```

[^1]: refer to [themes](./docs/themes.md)
