# What is xk-toolbox?
Xk-Toolbox is just a library of stuff.
It contains a library of different modules and styles that I stored/will store for other projects.
It contains:
 - Nodejs Modules/ Js Modules
 - CSS Stylesheets
 - json config files
## What does it do?
Well, it does not do anything aside for being a library for js/ nodejs and css styles.
The purpose of this package is so that I can store any modules/ styles that I might use later.
This one should be more organized than my other packages.

## How do I use it?
- Installing into your nodejs project:
  - I use visual studio to install packages on my projects, you can learn how to that through the link [`here`](https://learn.microsoft.com/en-us/visualstudio/javascript/npm-package-management?view=vs-2022)
- To learn how to install them locally, you can Checkout this link here :arrow_right: [NPM Documentation](https://docs.npmjs.com/downloading-and-installing-packages-locally)
- When acessing the css styles:
  - HTML: 
  ``` 
  <link rel="stylesheet" href="https://unpkg.com/xk-toolbox@<version>/<theme>/style.css"/>
  ```
  - javascript:
  ```
  <script src="https://unpkg.com/xk-toolbox@1.0.1/lib/<module-name>/index.js" type="module"></script>
  ```
